Bootstrap 4 - Buttons

Botões

Estilos de Botões

O bootstrap oferece quatro tipos de botões:

Button Outline

Os botões outline contém linhas apenas em suas bordas: btn-outline-*

Tamanho de Botões

O Bootstrap 4 oferece dois tamanho de botões: .btn-sm e .btn-lg

Botões em bloco

A class .btn-block faz com que o botão seja exibido em block (Neste caso, ocupará toda largura do container)

Active e Disabled

A class .active faz parecer com que o botão esteja pressionado e o atributo disabled faz com que o botão seja inclicável. A tag <a> não suporta o atributo disabled, então a class .disabled faz o link parecer desativado.

Link desativado

Botões com Spinner

Você pode adicionar spinners no botão (mais informações sobre spinners posteriormente).

Grupo de Botões

Para fazer um grupo de botões, é usado a class .btn-group

.btn-group.sm e lg

Botões e dropdown menu

Para juntar botões e dropdown menu, deve ser usado a class .dropdown-toggle e a propriedade data-toggle='dropdown'
