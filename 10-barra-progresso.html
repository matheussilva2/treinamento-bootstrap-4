Bootstrap 4 - Barra de Progresso

Barra de progresso básica

Uma barra de progresso básica é criada pelas classes .progress e dentro da div uma outra com .progress-bar. Você pode usar a propriedade CSS width para mudar a porcentagem da barra.

Altura da Barra

A altura da barra é definida pela propriedade CSS heigth (por padrão, ela vem em 16px).



Label de Barra de Progresso

Você pode colocar textos dentro da barra.

Barras coloridas

Por padrão, a barra tem um fundo azul. Você pode trocar pela class bg-*


Stripped Bars

São barras mais bonitinhas rs. Você pode fazer usando a class .progress-bar-striped


Barras animadas ♥

Para fazer barras animadas, é só usar a class .progress-bar-animated (Não é necessário usar bootstrap.js, jquery e popper)

Multiplas barras

Você pode criar várias barras dentro de um único div.progress
