Bootstrap 4 - Paginação

Paginação Básica

Para criar uma paginação básica, aplique a class .pagination ao <ul> e .page-item para cada <li> e dentro do <li> coloque a class .page-link dentro do elemento <a>

Breadcrumbs

Esta é uma outra forma de paginação, ela mostra a localização do usuário no site, baseado em uma hierarquia. Para usar, aplique a class .breadcrumb e .breadcrumb-item (O raciocínio é o mesmo de .pagination).
