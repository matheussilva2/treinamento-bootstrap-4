Grupo de listas

Grupo de Listas Básico

O tipo de grupo de lista mais básico é usando um <ul> e <li>. Para criar um grupo de listas, deve ser aplicado a class .list-group e .list-group-item.

Você pode aplicar a class .active para ativar um item

Grupo de lista com links

Para fazer este tipo de lista, deve aplicar a class .list-group-item a um elemento <a> juntamente com a class .list-group-item-action

Você pode aplicar a class .disabled para desativar um item.
