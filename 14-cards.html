Bootstrap 4 - Cards

O que são cards?

O card é uma caixa com bordas e padding ao redor de seu conteúdo. Esta class permite adição de header, footer e conteúdos.

Para criar um card básico, adicionamos a class .card a um div e .card-body em um div filho.

Exemplo 

<div class="card">
	<div class="card-body">Card básico</div>
</div>
Resultado
Card básico

Header, Body e Footer

Nós podemos estruturar um card através das classes: .card-header, .card-body e .card-footer.

Exemplo 

<div class="card">
	<div class="card-header">Cabeçalho</div>
	<div class="card-body">Corpo</div>
	<div class="card-footer">Rodapé</div>
</div>
Resultado
Cabeçalho
Corpo
Rodapé

Cards Contextuais

Podemos adicionar classes de background (.bg-*) para contextualizar os cards.

Clique aqui para ver sobre contextos.

Exemplo 

<div class="card bg-primary text-white">
	<div class="card-body">Card Primary</div>
</div>
<div class="card bg-warning text-white">
	<div class="card-body">Card Warning</div>
</div>
Resultado

Card Primary
Card Warning

Títulos, textos e links

Use a class .card-title em um elemento de cabeçalho. A class .card-text é usada para remover as margins inferiores se ele for o último elemento filho (ou o único) dentro de um .card-body. A class .card-link adiciona uma cor azul e um efeito de hover ao link.

Exemplo 

<div class="card">
	<div class="card-body">
		<h4 class="card-title">Título do Card</h4>
		<p class="card-text">Lorem ipsum dolor</p>
		<a class="card-link" href="#">Card Link</a>
		<a class="card-link" href="#">Outro link</a>
	</div>
</div>

Imagens nos Cards

Adicione a class .card-img-top ou .card-img-bottom em um <img> para adicionar a imagem na parte superior ou inferior do card. O elemento <img> deve ficar fora do body.

Exemplo 

<div class="card" style="width: 400px;">
	<img class="card-img-top" src="https://www.w3schools.com/bootstrap4/img_avatar1.png"/>
	<div class="card-body">
		<h4 class="card-title">John Doe</h4>
		<p class="card-text">Lorem ipsum dolor</p>
		<a class="btn btn-primary">Ver Perfil</a>
	</div>
</div>

Coluna de Cards

A class .card-columns cria uma grade com os cards (tipo Pinterest). O layout se ajusta automaticamente conforme você adiciona cards.

Nota: Os cards são exibidos verticalmente em pequenas telas (menor do que 576px).
