Bootstrap 4 - Dropdown

Dropdown Básico

Opção 1 Opção 2 Opção 3 

<div class="dropdown">
	<button type="button" class="btn btn-success dropdown-toggle" data-toggle="dropdown">Botão Dropdown</button>
	<div class="dropdown-menu">
		<a class="dropdown-item" href="#">Opção 1</a>
		<a class="dropdown-item" href="#">Opção 2</a>
		<a class="dropdown-item" href="#">Opção 3</a>
	</div>
</div>
  • A class .dropdown indica que ali será um menu dropdown.
  • Para abrir um dropdown menu, use a class .dropdown-toggle e o atributo data-toggle="dropdown"
  • Adicione a class .dropdown-menu em uma div para construir o menu e então adicione a class .dropdown-item para cada item.

Divisor dropdown

Use a class .dropdown-divider para criar uma linha horizontal com bordas para criar um divisor.
