Bootstrap 4 - Navs

Nav Menus

Se quer criar um simples menu horizontal, crie um lista desordenada <ul> com a class .nav, cada <li> com a class .nav-item e cada <a> com a class .nav-link

  • Item 1
  • Item 2
  • Item 3
  • Desativado
Código 
<ul class="nav">
	<li class="nav-item">
		<a class="nav-link" href="#">Item 1</a>
	</li>
	<li class="nav-item">
		<a class="nav-link" href="#">Item 2</a>
	</li>
	<li class="nav-item">
		<a class="nav-link" href="#">Item 3</a>
	</li>
	<li class="nav-item">
		<a class="nav-link disabled" href="#">Desativado</a>
	</li>
</ul>

Alinhando .nav

Você pode alinhar sua nav no meio ou à direita adicionando as classes .justify-content-center ou .justify-content-end na tag <ul>

Abas/Pílulas Dinâmicas

Para fazer abas que alternam, siga os seguintes passos:

  • Adicione o atributo data-toggle="tab" para cada link.
  • Adicione a class .tab-pane nas divs com id único, todas dentro de uma div com a class .tab-content
Se quiser fazer o efeito de fade-in/out, adicione a class .fade nas div com a class .tab-pane
