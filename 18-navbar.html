Navbars - Bootstrap 4

Navbar Básico

Siga os seguintes passos para criar um navbar básico:

  • Crie um div.navbar..navbar-expand.xl|lg|md|sm (o expand define em qual tamanho de tela o navbar vai se expandir. Se a tela for menor, ele vai collapsar)
  • Item 1
  • Item 2
  • Item 3
Código 
<div class="navbar navbar-expand-md bg-white">
	<ul class="navbar-nav">
		<li class="nav-item">
			<a href="#" class="nav-link">Item 1</a>
		</li>
		<li class="nav-item">
			<a href="#" class="nav-link">Item 2</a>
		</li>
		<li class="nav-item">
			<a href="#" class="nav-link">Item 3</a>
		</li>
	</ul>
</div>

Collapsando Navbar

Este formato de navbar é geralmente utilizados para telas pequenas.

  1. Crie um button.navbar-toggler com o atributo data-toggle="collapse" e data-target="#navbar-alvo"
  2. Adicione a class .collapse .navbar-collapse em uma div que isola a lista de links
Logomarca
  • Link 1
  • Link 2
  • Link 3
Código 
<nav class="navbar navbar-expand-md bg-dark navbar-dark">
	<!-- Logomarca -->
	<a href="#" class="navbar-brand">Logomarca</a>

	<!-- Botão para collapsar -->
	<button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navAlvo1">
		<span class="navbar-toggler-icon"></span>
	</button>

	<!-- Navbar links -->
	<div class="collapse navbar-collapse" id="navAlvo1">
		<ul class="navbar-nav">
			<li class="nav-item">
				<a href="#" class="nav-link">Link 1</a>
			</li>
			<li class="nav-item">
				<a href="#" class="nav-link">Link 2</a>
			</li>
			<li class="nav-item">
				<a href="#" class="nav-link">Link 3</a>
			</li>
		</ul>
	</div>
</nav>
