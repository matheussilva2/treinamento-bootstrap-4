Bootstrap 4

Formulários

Por padrão, o bootstrap trás estilizações para formulários. Os elementos input, textarea e select carregam a class .form-control

Layout de formulários

O bootstrap possui dois tipos de layout de formulários:

  • Formulário em empilhado (largura completo)
  • Linear

Formulário Empilhado

Neste estilo de formulário, crie um div.form-group. Cada .form-group indica um elemento da pilha.

Código 
<form action="#">
	<div class="form-group">
		<label for="email">Email: </label>
		<input type="email" class="form-control" id="email">
	</div>
	<div class="form-group">
		<label for="pwd">Senha: </label>
		<input type="password" class="form-control" id="pwd">
	</div>
	<div class="form-group form-check">
		<label class="form-check-label">
			<input type="checkbox" class="form-check-input"> Lembrar Senha
		</label>
	</div>
	<button type="submit" class="btn btn-primary">Submit</button>
</form>

Formulário Linear

Para fazer um formulário linear, adicione a class .form-inline.

OBS: Este layout apenas funciona em telas maiores do que 576px de largura. Em telas menores, o formulário será empilhado verticalmente.
Código 
<form action="#" class="form-inline">
	<label for="email">Email: </label>
	<input type="email" class="form-control" id="email">
	<label for="pwd">Senha: </label>
	<input type="password" class="form-control" id="pwd">
	<div class="form-check">
		<label class="form-check-label">
			<input type="checkbox" class="form-check-input">Lembrar Senha
		</label>
	</div>
	<button type="submit" class="btn btn-bootstrap text-white">Enviar</button>
</form>

Formulário Linear com Utilidades

O formulário linear parece um pouco "comprimido". No exemplo a seguir, adiciono uma margem à direita para todos os dispositivos e uma margem embaixo para caso não houver espaço suficiente, ser mostrado verticalmente (.mb-2 .mr-sm-2).

Código 
<form action="#" class="form-inline">
	<label for="email" class="mr-sm-2">Email: </label>
	<input type="email" id="email" class="form-control mb-2 mr-sm-2">
	<label for="pwd" class="mr-sm-2">Senha: </label>
	<input type="password" id="pwd" class="form-control mr-sm-2 mb-2">
	<div class="form-check mb-2 mr-sm-2">
		<label class="form-check-label">
			<input type="checkbox" class="form-check-input">Lembrar Senha
		</label>
	</div>
	<button class="btn btn-success mb-2">Enviar</button>
</form>
