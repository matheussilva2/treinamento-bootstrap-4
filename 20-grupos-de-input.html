Grupos de Input

Esta class contém um único raciocínio que irá acompanhar todo este artigo.

Para criar um grupo de input, use a class .input-group

Dentro deste grupo, vc pode colocar um texto ou icone de ajuda usando a class .input-group-prepend para ficar antes do input ou .input-group-append para ficar após o input.

Exemplos

@

Pesquisa

Código 
<form>
	<div class="input-group mb-3">
		<div class="input-group-prepend">
			<span class="input-group-text">@</span>
		</div>
		<input class="form-control" type="text" placeholder="email@server.com">
	</div>
	<div class="input-group mb-3">
		<input type="text" class="form-control" placeholder="Pesquisa">
		<div class="input-group-append">
			<p class="input-group-text">Pesquisa</p>
		</div>
	</div>
</form>

Dicazinha :3

Coloque um .mb-3 para dar uma borda legal.
