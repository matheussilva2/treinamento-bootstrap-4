Formularios Personalizados

O bootstrap trás classes para personalizar inputs de formulários, deixando-os diferente dos estilos padrões.

Criando Formulário Personalizados

Existe um padrão de criação de inputs personalizados. Para os elementos checkbox, switch e radio segue o seguinte padrão:

  • Crie uma div com a class .custom-control e .custom-control-*(input type)
  • Adicione a class .custom-control.input ao input
  • Se houver label, adicione a class .custom-control-label a ele.
Checkbox personalizado
<form action="#">
	<div class="custom-control custom-checkbox mb-3">
		<input id="checkbox1" type="checkbox" class="custom-control-input" name="exemplo1">
		<label for="checkbox1" class="custom-control-label"> Customizado</label>
	</div>
	<input type="checkbox">
	<label> Nativo</label>
</form>

Formularios Customizados Lineares

Para fazer isso basta você adicionar a class .custom-control-inline na div que está guardando os inputs.

Select Menu Customizado

Este é um dos inputs que não segue o padrão, porém é mais simplificado.

  • Apenas adicione a class .custom-select no elemento <select>

Dica

Você pode mudar o tamanho do select input através da class .custom-select-* lg/sm

Range Customizado

Apenas adicione a class .custom-range ao input do tipo range

Envio de Arquivo Customizado

Este é um pouco mais complexo, porém não é difícil.

  • Crie uma div com a class .custom-file
  • Adicione a class .custom-file-input no input.
  • Se for usar label, aplique a class .custom-control-label
Envio de Arquivo Customizado
<form>
	<div class="custom-file">
		<input id="customFile" type="file" class="custom-file-input">
		<label for="customFile" class="custom-file-label">Escolher Arquivo...</label>
	</div>
</form>

//Script para Atualizar o label com o nome do arquivo
<script type="text/javascript">
	$(".custom-file-input").on("change",function(){
		var nomeArquivo = $(this).val().split("\\").pop();
		$(this).siblings('.custom-file-label').addClass('selected').html(nomeArquivo);
	});
</script>
