Carousel

Carousel é um apresentação de slide no estilo carrossel(como o próprio nome diz). A estrutura de um carousel é bem estruturada e simples de compreensão. Confira o exemplo: 

<div id="demo" class="carousel slide" data-ride="carousel">
	<!-- Indicadores -->
	<ul class="carousel-indicators">
		<li data-target="#demo" data-slide-to=0 class="active"></li>
		<li data-target="#demo" data-slide-to=1></li>
		<li data-target="#demo" data-slide-to=2></li>
	</ul>

	<!-- As imagens -->
	<div class="carousel-inner">
		<div class="carousel-item">
			<img src="https://www.w3schools.com/bootstrap4/la.jpg">
		</div>
		<div class="carousel-item">
			<img src="https://www.w3schools.com/bootstrap4/chicago.jpg">
		</div>
		<div class="carousel-item">
			<img src="https://www.w3schools.com/bootstrap4/ny.jpg">
		</div>
	</div>

	<!-- Controles Avançar e Retroceder -->
	<a class="carousel-control-prev" href="#demo" data-slide="prev">
		<i class="carousel-control-prev-icon"></i>
	</a>
	<a class="carousel-control-next" href="#demo" data-slide="next">
		<i class="carousel-control-next-icon"></i>
	</a>
</div>

Tabela com cada class

Class Descrição
.carousel Cria o carousel
.carousel-indicators Adiciona indicadores que te permite pular para um item especifico e também mostra o item atual
.carousel-item Cria um item para o slide
.carousel-control-prev Adiciona um botão para voltar ao o item anterior do carousel.
.carousel-control-next Adiciona um botão para avançar ao próximo item do carousel.
.slide Adiciona a transição deslizante dos itens. Remova essa class se você não quiser esta transição.

Legendas

Você pode adicionar legenda a cada item do seu slide, apenas adicione um div.carousel-caption

<div class="carousel-item">
  <img src="link-da-img"">
  <div class="carousel-caption">
    <h3>Título</h3>
    <p>Sua descrição entra aqui!</p>
  </div>
</div>
